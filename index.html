🎯 개요

로마서 8장 39절을 구절 단위로 반복 입력하며 암송하는 도구입니다. 각 구절에서 일부 단어가 빈칸으로 가려지고, 정답을 맞히면 다음 구절로 넘어갑니다.

⚙️ 설정 화면

  • 시작 / 끝 구절: 학습할 범위 (1~39절)
  • 숨기기 기능: 문제 노출 후 2초 대기, 이후 서서히 구절 전체가 흐려져 사라짐
  • 텍스트 입력: 키보드로 전체 구절을 타이핑할지 여부 (끄면 힌트/전체 보기만으로 연습)
  • 문제 보는 시간: 숨기기 전까지 총 노출 시간(초)
  • 난이도 레벨: Lv.1(10%) ~ Lv.10(100%) — 빈칸 처리되는 단어 비율
  • 연속 정답 횟수: 이 횟수만큼 연속 정답해야 다음 구절로 이동

📝 테스트 화면

  • 제출: 입력한 전체 구절을 채점 — 일치하면 파란색으로 표시 후 다음 단계로 이동
  • 💡 힌트: 누를 때마다 빈칸 중 하나를 랜덤 위치로 즉시 공개 (숨김 ON/OFF 무관하게 계속 누를 수 있음)
  • 👁 전체 보기 / 🙈 전체 숨기기: 빈칸을 포함한 구절 전체 토글
  • ⌨ 입력 ON/OFF: 타이핑 영역 표시 여부 토글
  • ◀ Prev / Next ▶: 구절 수동 이동 (연속 카운트 초기화)
  • 종료: 설정 화면으로 돌아감

⌨ 키보드 단축키

  • Enter — 제출 (입력 칸에 포커스 있을 때)
  • Shift+Enter — 줄바꿈
  • Home — 전체 보기 ↔ 전체 숨기기 토글
  • End — 힌트 (매번 랜덤 위치)
  • PageDown / + / 마우스 휠 ↓ — 다음 구절
  • PageUp / - / 마우스 휠 ↑ — 이전 구절
