개역개정 · 직접 입력 하면서 외우기

프로그램 사용 설명

🎯 개요

개역개정 로마서 8장(1~39절)을 구절 단위로 반복 학습하며 암송하는 도구입니다. 각 구절에서 난이도에 따라 일부 단어가 빈칸으로 가려지고, 힌트·낭독·전체보기 등을 활용해 구절을 익힙니다.

⚙️ 설정 화면

  • 시작 / 끝 구절: 학습할 범위 (1~39절)
  • 숨기기 기능 (기본 OFF): ON일 때 문제가 2초 노출 후 서서히 흐려져 사라짐. OFF면 전체·힌트·오답 화면도 자동 숨김 동작이 모두 멈춥니다.
  • 텍스트 입력 (기본 OFF): 키보드로 전체 구절을 타이핑하여 제출할지 여부. OFF면 입력 영역과 제출 버튼이 숨겨지고 힌트·전체 보기·낭독으로만 연습.
  • 문제 보는 시간 (기본 30초): 숨기기 ON일 때 노출 → 페이드아웃 → 자동 진행까지 걸리는 총 시간.
  • 난이도 레벨 (기본 Lv.1): Lv.1(10%) ~ Lv.10(100%). 단어 중 해당 비율이 빈칸으로 바뀜.
  • 연속 정답 횟수 (기본 1회): 이 횟수만큼 연속 정답해야 다음 구절로 이동.
  • 설정 초기화: 저장된 설정을 모두 삭제하고 페이지를 새로고침해 기본값으로 복원.

📝 테스트 화면 버튼

  • 제출: 입력한 전체 구절을 채점. 정답이면 구절 전체가 파랗게 강조된 뒤 자동으로 다음 단계로 이동, 오답이면 틀린 단어가 표시됩니다.
  • 💡 힌트: 빈칸들을 셔플 큐 방식으로 한 싸이클 동안 각 1회씩 균등하게 공개. 싸이클이 끝나면 다시 셔플해서 새 싸이클 시작.
  • 🔊 낭독: 현재 구절을 한국어 음성으로 읽어줍니다. 재클릭 시 중지 (브라우저/OS 음성 지원 필요).
  • 👁 전체 보기 / 🙈 전체 숨기기: 빈칸 포함 여부 토글. 숨기기 OFF 상태에서는 "전체 숨기기"가 빈칸 있는 원래 상태로 복귀.
  • ⌨ 입력 ON / OFF: 타이핑 영역과 제출 버튼의 표시 여부 토글.
  • ◀ Prev / Next ▶: 구절 수동 이동 (연속 정답 카운트 초기화).
  • 종료: 진행 중인 테스트를 중단하고 설정 화면으로 복귀.

🟢 오답 직후 동작

틀리면 정답 구절이 표시되며 입력창은 즉시 다시 활성화됩니다 — 노출 대기 시간 동안에도 바로 다음 시도를 입력할 수 있어요. 자동 다음 이동은 문제 보는 시간 후에 일어납니다.

⌨ 키보드 / 마우스 단축키

  • Enter — 제출 (입력 칸에 포커스 있을 때)
  • Shift+Enter — 입력 줄바꿈
  • Home — 전체 보기 ↔ 전체 숨기기 토글
  • End — 힌트 (셔플 큐 다음 빈칸)
  • PageDown / + · = / 마우스 휠 ↓ — 다음 구절
  • PageUp / - · _ / 마우스 휠 ↑ — 이전 구절
  • Esc — 이 도움말 창 닫기

📱 설치 (PWA)

크롬/에지/사파리의 "홈 화면에 추가"(모바일) 또는 주소창 설치 아이콘으로 앱처럼 설치할 수 있고, 한 번 열어본 뒤에는 오프라인에서도 실행됩니다.

💾 저장

설정값과 테마는 브라우저 localStorage에 자동 저장되어 다음 방문 시 복원됩니다. 초기화하려면 설정 화면의 설정 초기화 버튼을 눌러주세요.
