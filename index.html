개역개정 · 직접 입력 하면서 외우기 · v0.0.6

프로그램 사용 설명

🎯 개요

개역개정 성경의 선택한 본문(현재 로마서 8장, 고린도전서 13장)을 구절 단위로 반복 학습하며 암송하는 도구입니다. 각 구절에서 난이도에 따라 일부 단어가 빈칸으로 가려지고, 힌트·낭독·전체보기 등을 활용해 구절을 익힙니다.

⚙️ 설정 화면

  • 본문 선택: 학습할 성경 본문(장 단위). 선택에 따라 시작/끝 구절의 허용 범위가 자동 조정됩니다.
  • 시작 / 끝 구절: 선택한 본문 내에서 학습할 범위.
  • 텍스트 입력 (기본 OFF): 키보드로 전체 구절을 타이핑하여 제출할지 여부. OFF면 입력 영역과 제출 버튼이 숨겨지고 힌트·전체 보기·낭독으로만 연습.
  • 구절 이동 시 자동 전체보기 (기본 OFF): ON이면 수동으로 구절을 이동할 때(◀/▶ 버튼, 화살표, 휠, PageUp/Down) 현재 구절의 빈칸을 전체 단어 수 × 1초 동안 정답으로 채워 미리보고 난 뒤 다음 구절로 넘어감 (예: 단어 12개 → 12초). 정답/오답으로 자동 이동할 때는 이미 답이 공개되므로 별도 미리보기 없음.
  • 문제 보는 시간 (기본 30초): 정답 시 전체 정답 표시, 오답 시 정답 하이라이트 화면을 유지한 뒤 자동으로 다음 단계로 진행하기까지의 대기 시간.
  • 난이도 레벨 (기본 Lv.1): Lv.1(10%) ~ Lv.10(100%). 단어 중 해당 비율이 빈칸으로 바뀜.
  • 연속 정답 횟수 (기본 1회): 이 횟수만큼 연속 정답해야 다음 구절로 이동.
  • 낭독 음성: 🔊 낭독에 사용할 TTS 음성. 자동 모드는 신경망/클라우드 음성을 우선 고르며, ⭐ 표시는 신경망/Natural/Premium 계열 고품질 음성입니다.
  • 낭독 속도: 0.5(느림) ~ 1.5(빠름), 기본 0.95.
  • 설정 초기화: 저장된 설정을 모두 삭제하고 페이지를 새로고침해 기본값으로 복원.

🎙 고품질 낭독 음성 가이드

가장 자연스러운 한국어 음성을 원한다면 다음 환경을 권장합니다:

  • Microsoft Edge: "Microsoft SunHi/InJoon Online (Natural)" 음성 — 가장 자연스러움
  • macOS / iOS Safari: 설정 → 손쉬운 사용 → 음성에서 Yuna (고급/Premium) 다운로드
  • Android Chrome: Google TTS 엔진의 "한국어 (대한민국)" 음성 (WaveNet 계열)
  • Windows: 설정 → 시간 및 언어 → 음성 → 한국어 "향상된 자연스러운 음성" 설치

설치 후 페이지를 새로고침하면 "낭독 음성" 드롭다운에서 ⭐ 표시로 자동 인식됩니다.

📝 테스트 화면 버튼

  • 제출: 입력한 전체 구절을 채점. 정답이면 구절 전체가 파랗게 강조된 뒤 자동으로 다음 단계로 이동, 오답이면 틀린 단어가 표시됩니다.
  • 💡 힌트 보기 / 힌트 숨기기: 토글 버튼. "보기"를 누르면 셔플 큐의 다음 빈칸 하나를 공개하고, "숨기기"를 누르면 즉시 감춥니다. 셔플 큐는 한 싸이클 동안 각 빈칸이 1회씩 균등하게 공개되도록 동작합니다.
  • 🔊 낭독: 현재 구절을 한국어 음성으로 읽어줍니다. 재클릭 시 중지 (브라우저/OS 음성 지원 필요).
  • 👁 전체 보기 / 🙈 전체 숨기기: 빈칸 포함 여부 토글. "전체 보기"는 모든 빈칸을 정답으로 채우고, "전체 숨기기"는 다시 빈칸 상태로 되돌립니다.
  • 📖 자동보기 ON / OFF: 수동 구절 이동 직전 현재 구절의 전체 단어 수만큼(초) 정답을 미리 보여줄지 토글. 빠르게 다시 이동을 누르면 미리보기는 건너뛰고 바로 이동합니다.
  • ⌨ 입력 ON / OFF: 타이핑 영역과 제출 버튼의 표시 여부 토글.
  • ◀ Prev / Next ▶: 구절 수동 이동 (연속 정답 카운트 초기화).
  • 종료: 진행 중인 테스트를 중단하고 설정 화면으로 복귀.

🟢 정답 / 오답 직후 동작

정답: 구절 전체 단어가 파랗게 강조된 상태로 문제 보는 시간만큼 유지된 뒤, 연속 정답 횟수를 만족하면 다음 구절로 넘어갑니다.

오답: 정답 구절이 표시되고 틀린 단어에 빨간 밑줄이 그려집니다. 입력창은 즉시 다시 활성화되므로 대기 시간 동안에도 바로 다음 시도를 입력할 수 있고, 자동 재시도 화면으로의 이동은 문제 보는 시간 이후에 일어납니다.

⌨ 키보드 / 마우스 단축키

  • Enter — 제출 (입력 칸에 포커스 있을 때)
  • Shift+Enter — 입력 줄바꿈
  • Home — 전체 보기 ↔ 전체 숨기기 토글
  • End — 힌트 (셔플 큐 다음 빈칸)
  • PageDown / + · = / 마우스 휠 ↓ — 다음 구절
  • PageUp / - · _ / 마우스 휠 ↑ — 이전 구절
  • → / ← — 다음 / 이전 구절 (입력 OFF이거나 입력 칸에 포커스 없을 때)
  • Enter (설정 화면) — 테스트 시작
  • Esc — 이 도움말 창 닫기

📱 설치 (PWA)

크롬/에지/사파리의 "홈 화면에 추가"(모바일) 또는 주소창 설치 아이콘으로 앱처럼 설치할 수 있고, 한 번 열어본 뒤에는 오프라인에서도 실행됩니다.

💾 저장 및 버전 관리

설정값은 브라우저 localStorage에 자동 저장되어 다음 방문 시 복원됩니다. 초기화하려면 설정 화면의 설정 초기화 버튼을 눌러주세요.

앱 버전이 올라가면 기본값/스키마 충돌을 막기 위해 저장된 설정이 자동으로 초기화됩니다 — 새 버전이 로드되면 저장된 값은 버려지고 기본값으로 시작합니다.
