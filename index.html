Bible Memorization

개역개정 · 직접 입력 하면서 외우기 · v0.0.60

프로그램 사용 설명

🎯 개요

개역개정 성경 본문을 구절 단위로 반복 학습하며 암송하는 도구입니다. 본문은 세 분류로 제공됩니다:

  • 성경 — 개역개정 전체 66권 (책 선택 후 장 선택)
  • 훈련 교재 — 로마서 8장(사역훈련 1과), 사역훈련 2권(14과), 사역훈련 3권(8과), 제자훈련 1권(6과), 제자훈련 2권(14과), 제자훈련 3권(12과)
  • Bible Memory — 고린도전서 13장, 시편 1편

각 구절에서 난이도에 따라 일부 단어가 빈칸으로 가려지고, 힌트·전체보기·북마크·북마크 필터 등을 활용해 구절을 익힙니다. 빈칸 위치는 매 테스트 시작마다 암호학적 난수로 새로 추첨됩니다. 시작 화면에서 오프라인 필기 연습용 빈칸 PDF도 바로 생성할 수 있습니다.

⚙️ 설정 화면

  • 분류 (기본 훈련 교재): 성경 / 훈련 교재 / Bible Memory 중 하나. 선택에 따라 아래 본문 선택 드롭다운이 필터링됩니다.
  • 본문 선택: 현재 분류 안의 암송 본문. 분류가 성경이면 아래에 장 선택이 추가로 나타나고, 훈련 교재 / Bible Memory는 과 또는 장 단위로 바로 선택됩니다.
  • 시작 / 끝 구절 (또는 과): 선택한 본문 내에서 학습할 범위. 훈련 교재(레슨 단위)는 과, 나머지는 절 단위.
  • 텍스트 입력 (기본 OFF): 키보드로 전체 구절을 타이핑하여 제출할지 여부. OFF면 입력 영역과 제출 버튼이 숨겨지고 힌트·전체 보기로만 연습.
  • 구절 이동 시 자동 전체보기 (기본 ON): ON이면 수동으로 구절을 이동할 때(◀/▶ 버튼, 화살표, PageUp/Down) 현재 구절의 빈칸을 전체 단어 수 × 1초 동안 정답으로 채워 미리보고 난 뒤 다음 구절로 넘어감 (예: 단어 12개 → 12초). 정답/오답으로 자동 이동할 때는 이미 답이 공개되므로 별도 미리보기 없음. 이전 구절(◀ Prev)로 이동할 때는 자동보기가 동작하지 않습니다.
  • 처음 두 단어 제약 (기본 없음): 구절 첫 두 단어를 빈칸 처리하는 방식. 없음 — 제약 없이 난이도 비율로만 무작위 빈칸. 처음 두 단어만 보이기 — 첫 두 단어만 보이고 나머지 전부 빈칸(회상 훈련). 처음 두 단어만 가리기 — 첫 두 단어만 빈칸, 나머지는 모두 보임. 처음 두 단어도 가리기 — 난이도가 정한 빈칸 개수 안에서 첫 두 단어를 우선 선택(레벨 0개면 빈칸 없음). 처음 두 단어는 반드시 가리기 — 난이도와 무관하게 첫 두 단어 항상 빈칸 + 나머지는 레벨 비율로 추가.
  • 연속 빈칸 하나로 합치기 (기본 OFF): ON이면 서로 인접한(공백만 사이에 둔) 빈칸들을 하나의 긴 빈칸으로 묶어 표시. 어디가 몇 단어인지까지 추측해야 하므로 난이도가 체감상 상승합니다. 힌트로 가운데 단어가 공개되면 해당 위치에서 자동으로 분할됩니다. 채점 로직은 동일(단어 단위).
  • 난이도 레벨 (기본 Lv.6): Lv.1(10%) ~ Lv.10(100%). 단어 중 해당 비율이 빈칸으로 바뀜. Lv.7 이상이면 상단 본문 제목/참조도 단계적으로 가려집니다(Lv.7 제목 숨김 · Lv.8 참조 숨김 · Lv.9~10 둘 다 숨김).
  • 연속 정답 횟수 (기본 1회): 이 횟수만큼 연속 정답해야 다음 구절로 이동.
  • 북마크된 구절만 (기본 OFF): ON이면 현재 본문에서 북마크된(⭐) 구절만 모아 번호 순서대로 테스트. 북마크가 없으면 알림 후 시작되지 않습니다.
  • 📄 오프라인 연습지 인쇄 (PDF): 시작 화면 하단의 PDF 섹션은 아래 📄 오프라인 연습지 인쇄 (PDF) 항목을 참고.
  • 북마크 전체 삭제: 모든 본문의 저장된 북마크를 일괄 삭제. 북마크는 오직 이 버튼으로만 삭제 가능하며, 설정 초기화·앱 버전 업데이트와 무관하게 영구 보존됩니다.
  • 설정 초기화: 저장된 설정을 모두 삭제하고 페이지를 새로고침해 기본값으로 복원. 북마크는 영향받지 않습니다.

📄 오프라인 연습지 인쇄 (PDF)

프린터로 출력하거나 브라우저의 "PDF로 저장"을 통해 오프라인 필기 연습용 빈칸 시험지를 만드는 기능입니다. 시작 화면에서만 사용 가능합니다.

  • 범위: 현재 설정된 시작 ~ 끝 구절(또는 과) 전체. 단, 북마크된 구절만이 ON이면 해당 본문의 북마크(⭐) 구절만 대상.
  • Set 수 (1 ~ 10, 기본 3): 한 PDF 안에 포함될 시험 세트 수. 각 세트는 새 페이지로 시작하며, 같은 난이도·설정 안에서 빈칸 위치가 매번 새로 무작위 추첨되어 서로 다른 시험지 효과를 냅니다.
  • 빈칸 스타일: 단어 길이 맞춤(빈칸 밑줄 길이가 원래 단어 길이에 비례 → 힌트 제공) / 고정 길이(모든 단어 빈칸이 동일한 기본 너비 → 추가 힌트 최소). 병합된 빈칸은 자동으로 더 넓게 표시.
  • 난이도 · 첫 두 단어 제약 · 연속 빈칸 합치기: 위 설정 화면의 값이 그대로 반영됩니다.
  • 정답지: PDF에 포함되지 않습니다(사용자 요청으로 생략).
  • 인쇄 팁: 브라우저 인쇄 대화상자에서 「여백」을 「좁게」 / 「최소」 / 「없음」으로 설정하면 한 페이지에 더 많은 구절이 들어갑니다. CSS 기본 여백은 10mm이지만 브라우저 설정으로 override 가능합니다.

📝 테스트 화면 버튼

  • 제출: 입력한 전체 구절을 채점. 정답이면 구절 전체가 파랗게 강조된 뒤 자동으로 다음 단계로 이동, 오답이면 틀린 단어가 표시됩니다.
  • 💡 힌트 보기 / 힌트 숨기기: 토글 버튼. "보기"를 누르면 셔플 큐의 다음 빈칸 하나를 공개하고, "숨기기"를 누르면 즉시 감춥니다. 셔플 큐는 한 싸이클 동안 각 빈칸이 1회씩 균등하게 공개되도록 동작합니다.
  • ☆ / ⭐ 북마크: 현재 구절을 북마크에 추가/해제 (토글). 북마크는 본문별로 저장되며, 설정 화면의 북마크된 구절만 ON 상태로 다시 시작하면 해당 구절만 모아 테스트할 수 있습니다. 테스트 중 Shift 단독 키로도 토글 가능.
  • 🔖 북마크만 ON / OFF: 테스트 도중 북마크 필터를 즉시 전환. ON으로 전환하면 현재 본문의 북마크 구절만 모은 목록으로 바뀌고, OFF로 돌리면 원래 시작·끝 구절 범위로 복귀합니다. 현재 구절이 새 목록에 없으면 가장 가까운 앞 구절로 이동합니다.
  • 👁 전체 보기 / 🙈 전체 숨기기: 빈칸 포함 여부 토글. "전체 보기"는 모든 빈칸을 정답으로 채우고, "전체 숨기기"는 다시 빈칸 상태로 되돌립니다.
  • 🔀 빈칸 섞기: 현재 구절의 빈칸 위치를 즉시 다시 추첨. 같은 난이도 내에서 다른 단어를 연습하고 싶을 때 사용 (Ctrl 단독 탭과 동일).
  • 📖 자동보기 ON / OFF: 수동 구절 이동 직전 현재 구절의 전체 단어 수만큼(초) 정답을 미리 보여줄지 토글. 빠르게 다시 이동을 누르면 미리보기는 건너뛰고 바로 이동합니다. 이전 구절(◀ Prev)로 이동할 때는 자동보기가 동작하지 않습니다.
  • ⬆ Lv+ / ⬇ Lv−: 테스트 중 난이도를 1~10 범위에서 즉시 변경하고 현재 구절의 빈칸을 새 비율로 다시 추첨. 변경된 난이도는 설정에 저장되어 다음 테스트에도 유지됩니다.
  • ⌨ 입력 ON / OFF: 타이핑 영역과 제출 버튼의 표시 여부 토글.
  • ◀ Prev / Next ▶: 구절 수동 이동 (연속 정답 카운트 초기화).
  • 종료: 진행 중인 테스트를 중단하고 설정 화면으로 복귀.

🟢 정답 / 오답 직후 동작

정답: 구절 전체 단어가 파랗게 강조된 상태로 30초간 유지된 뒤, 연속 정답 횟수를 만족하면 다음 구절로 넘어갑니다.

오답: 정답 구절이 표시되고 틀린 단어에 빨간 밑줄이 그려집니다. 입력창은 즉시 다시 활성화되므로 대기 시간 동안에도 바로 다음 시도를 입력할 수 있고, 자동 재시도 화면으로의 이동은 30초 뒤에 일어납니다.

⌨ 키보드 / 마우스 단축키

  • Enter — 제출 (입력 칸에 포커스 있을 때)
  • Shift+Enter — 입력 줄바꿈
  • Home — 전체 보기 ↔ 전체 숨기기 토글
  • End — 힌트 (셔플 큐 다음 빈칸)
  • PageDown / + · = — 다음 구절
  • PageUp / - · _ — 이전 구절
  • → / ← — 다음 / 이전 구절 (입력 OFF이거나 입력 칸에 포커스 없을 때)
  • ↑ / ↓ — 난이도 올리기 / 내리기 (입력 OFF이거나 입력 칸에 포커스 없을 때)
  • Shift (단독) — 현재 구절 북마크 토글 (입력 칸에 포커스 없을 때)
  • Ctrl (단독 탭) — 현재 구절 빈칸 위치 다시 뽑기
  • Enter (설정 화면) — 테스트 시작
  • Esc — 이 도움말 창 닫기

📱 설치 (PWA)

크롬/에지/사파리의 "홈 화면에 추가"(모바일) 또는 주소창 설치 아이콘으로 앱처럼 설치할 수 있고, 한 번 열어본 뒤에는 오프라인에서도 실행됩니다.

💾 저장 및 버전 관리

설정값은 브라우저 localStorage에 자동 저장되어 다음 방문 시 복원됩니다. 초기화하려면 설정 화면의 설정 초기화 버튼을 눌러주세요.

앱 버전이 올라가면 기본값/스키마 충돌을 막기 위해 저장된 설정이 자동으로 초기화됩니다 — 새 버전이 로드되면 저장된 값은 버려지고 기본값으로 시작합니다. 단, 북마크(⭐)는 버전 업데이트에도 영구 보존되며, 오직 설정 화면의 북마크 전체 삭제 버튼으로만 지울 수 있습니다.
