# 구약 성경 연대기 타임라인

> 창세기부터 말라기까지, 구약 39권을 가로축(시간축) 기준으로 배치한 연대기 자료입니다.
> 본 문서는 HTML 타임라인 페이지 제작을 위한 기초 데이터로 사용됩니다.

## 📌 문서 개요

- **대상**: 구약 성경 39권 (창세기 ~ 말라기)
- **정렬 기준**: 기록된 사건의 발생 시기 (BC 연대순)
- **표현 방식**: 가로축 타임라인 (왼쪽 = 가장 오래된 사건, 오른쪽 = 가장 최근 사건)
- **구분**: 역사서 / 시가서 / 예언서 (색상으로 구분)

## 🧭 HTML 타임라인 구현 가이드

### 권장 레이아웃 구조
```
[범례: 🟦 역사서  🟨 시가서  🟥 예언서]

◀─────────────────── 가로 스크롤 타임라인 ───────────────────▶
BC 2000   1500   1000   900   800   700   600   500   400
  │        │      │      │     │     │     │     │     │
 [창세기][출애굽기][다윗][솔로몬][이사야][예레미야][다니엘][말라기]
```

### 데이터 속성 제안 (각 책 블록에 적용)
- `data-book`: 책 이름 (예: "창세기")
- `data-category`: 분류 ("history" / "poetry" / "prophecy")
- `data-start-year`: 시작 연대 (음수로 BC 표현, 예: -2000)
- `data-end-year`: 종료 연대
- `data-color`: 분류별 색상 코드
- `data-summary`: 호버 시 표시될 요약

### CSS 색상 변수 예시
```css
:root {
  --color-history: #3B7EA1;   /* 역사서 */
  --color-poetry:  #E0A93A;   /* 시가서 */
  --color-prophecy: #B2383A;  /* 예언서 */
  --color-history-light: #D6E4EC;
  --color-poetry-light:  #F7E8C2;
  --color-prophecy-light: #EBCACB;
}
```

### 시각화 권장 사항
1. **가로축**: 로그 스케일보다는 **선형 스케일**이 직관적 (단, BC 2000 이전은 압축 표기 권장)
2. **책 블록**: 기간이 있는 책(예: 사사기 1375~1050)은 **막대(bar) 형태**, 단일 시점의 책은 **점(dot)**으로 표시
3. **겹침 처리**: 같은 시기의 책은 **세로로 쌓기(stacking)** — 역사서는 상단, 예언서는 하단 레인(lane) 등으로 구분
4. **인터랙션**: 호버 시 요약 툴팁, 클릭 시 상세 정보 패널
5. **반응형**: 모바일에서는 세로 스크롤 타임라인으로 전환 고려

## 📝 참고 사항

- 연대 추정치는 학자마다 차이가 있으며, 본 문서는 **보수적 복음주의 연대관**을 따랐습니다.
- 일부 책(특히 욥기, 오바댜, 요엘)의 연대는 학자 간 논쟁이 있습니다.
- 시편은 작성 기간이 매우 길어(다윗 시대 ~ 포로 귀환) 별도 시각화 고려가 필요합니다.
- **기록 시점**과 **사건 시점**이 다른 책(예: 역대기는 포로 귀환 이후 기록되었으나 다루는 사건은 아담부터)에 주의하세요.
