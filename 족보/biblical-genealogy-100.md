# 구약 주요 인물 100인 족보도 (예수 그리스도 족보 기준)

## 개요

이 문서는 마태복음 1장·누가복음 3장에 기록된 **아담 → 예수 그리스도의 메시아 계보**를 중심 뼈대(backbone)로 삼고, 각 시대마다 구약의 핵심 인물들을 가지(branch) 형태로 연결해 구속사의 큰 그림을 한눈에 볼 수 있도록 정리한 것이다.

### 선정 기준 (100인)

1. **성경 내 비중**: 독립 서사를 가지거나 다수 장(章)에 걸쳐 등장
2. **신학적 중요성**: 언약, 구원사, 예표론에서 핵심 역할
3. **인용 빈도**: 기독교 설교·신학 저서·교리서에서 반복 인용
4. **족보 직결성**: 메시아 계보와 직·간접 관계

### 범례 (Legend)

| 기호 | 의미 |
|------|------|
| **★** | 예수 족보 직계 (backbone, 61명 중 구약 인물 58명) |
| ● | 주요 인물 (branch 인물) |
| ├─ / └─ | 직계 자손 (혈연) |
| ╟─ | 배우자 / 중요 관계 |
| ⟶ | 영적 계승·후계 관계 |
| [ ] | 주계보 포지션 번호 |

## 신학적 의미

1. **구속사의 연속성**: 100인의 인물은 독립된 이야기가 아니라 "아담의 후손" → "아브라함의 언약" → "다윗의 왕권" → "메시아의 성취"라는 하나의 큰 흐름 위에 놓여 있다.

2. **언약의 점진적 계시**: 노아 언약(창 9) → 아브라함 언약(창 15, 17) → 모세 언약(출 19-24) → 다윗 언약(삼하 7) → 새 언약(렘 31) → 예수 그리스도에서 성취(히 8:6-13).

3. **은혜의 계보**: 주계보에 기생(라합), 이방 여인(룻), 간음(밧세바), 불의(유다·다말)가 등장한다는 사실은 메시아 구원의 보편성과 은혜성을 보여준다.

4. **예표론(Typology)**: 아담(둘째 아담), 노아(물 심판/구원), 이삭(모리아 번제), 요셉(고난받는 구속자), 모세(중보자), 여호수아(예수 이름의 원형, Yeshua), 멜기세덱(영원한 제사장), 다윗(메시아의 왕), 요나(삼일 부활) 등이 모두 예수 그리스도를 예표한다.

5. **실패와 회복**: 아담이 타락한 곳에서 예수께서 승리하셨고(롬 5), 이스라엘이 광야에서 실패한 시험을 예수께서 광야에서 이기셨다(마 4). 100인의 이야기는 인간의 실패와 하나님의 신실하심의 대비이다.

## 사용 예시

- 설교·성경공부에서 인물 관계와 시대 배치 빠른 확인
- "OO는 다윗부터 몇 대손인가" 같은 족보 거리 계산
- 마태복음 1장 vs 누가복음 3장 족보 비교 해설
- 예표론·언약신학 관점의 인물 분류
- 주일학교 교재 작성 시 시대별 정리
