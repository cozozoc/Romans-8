난이도는 level 1 부터 level 5까지 있어.
level 1 : 전체 단어 개수 중에 20% 가 빈칸
level 2 : 전체 단어 개수 중에 40% 가 빈칸
level 3 : 전체 단어 개수 중에 60% 가 빈칸
level 4 : 전체 단어 개수 중에 80% 가 빈칸
level 5 : 전체 단어 개수 중에 100% 가 빈칸

빈칸은 문제가 주어질 때 마다 위치가 랜덤하게 위치가 바뀌어야 해.
입력은 사용자로부터 받으며, 정답을 입력하면 다음 문제로 넘어가고, 오답을 입력하면 다시 시도할 수 있도록 해야 해. 또한, 사용자가 원하는 난이도를 선택할 수 있도록 해야 해.
프로그램은 사용자가 암송을 완료할 때까지 계속되어야 해.

입력은 빈칸에 입력하는 것이 아니라 전체 구절을 입력하는 방식으로 진행되어야 해. 즉, 빈칸이 많아져도 사용자가 전체 구절을 정확하게 다 입력하는가를 확인하는 시험이야.
빈칸 포함 문제로 주어지는 구절은 보여지고 나서 n 초가 지나면 사라져야 해. 사용자가 입력할 때는 빈칸이 포함된 구절이 보이지 않아야 해. 사용자가 입력한 구절이 정답과 일치하는지 확인하는 로직도 필요해.

첫 화면에 테스트 설정 화면이 떠야해.
시작 구절과 끝 구절을 입력 받아. default은 시작 구절은 1, 끝 구절은 39야.
숨기는 기능은 On/Off 설정이 있고 default on이야.
숨기는 시간은 초단위로 설정 가능하고 default는 3초야.

start level은 사용자가 설정할 수 있어. default은 3이야.
target level은 다음 구절로 넘어갈 목표 레벨이야. 목표 레벨에 도달해서 n번 연속 답을 맞추면 다름 구절로 넘어가. default target level은 5야 
연속 n번 맞추면 level이 높어지고 연속 3번 맞추면 level이 낮아져. default continuous correct count는 3이야.
시작할 설정한 level은 시험 진행 중 다음 구절로 넘어갈 때도 시작 level이야.
